# School Management System Frontend

Angular 18 application for School Management System with Bengali language support.

## Setup

1. **Install dependencies**:
```bash
npm install
```

2. **Development server**:
```bash
ng serve
```
Navigate to http://localhost:4200

3. **Build for production**:
```bash
ng build
```

## Features

- Bengali language interface
- Student management
- Attendance tracking
- Payment management
- Dashboard with statistics
- Responsive design

## Default Login
- Username: admin
- Password: password

## Components

- Login Component
- Dashboard Component
- Students Management
- Attendance Management
- Payments Management

## Services

- AuthService - Authentication
- StudentService - Student operations
- HTTP interceptors for API calls
